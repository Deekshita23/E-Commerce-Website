# 🛒 ShopEasy - E-Commerce Website

ShopEasy is a simple and responsive E-Commerce website built using **HTML, CSS, and JavaScript**.  
It allows users to browse products, search for items, and add products to the cart.

---

## 🚀 Features

- Responsive Design 📱💻
- Product Search Functionality 🔍
- Add to Cart System 🛒
- Modern Product Cards UI ✨
- Hover Effects & Smooth Styling 🎨

---

## 🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript (Vanilla JS)
